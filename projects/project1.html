Real-Time Event Tabulation System

A production-used system for live judged events, providing real-time score aggregation, automated ranking, and instant result generation.

Includes both system screenshots and photos/videos from actual events where the system was used.

This project is a real-time tabulation system developed to support live judged events organized by the HR department. It supports multiple judges scoring multiple contestants simultaneously, with results computed and displayed instantly.

Judges submit scores through a web interface, while an admin dashboard receives live updates via WebSockets. The system supports both point-based and rank-based scoring models to accommodate different competition formats.

Built with Laravel, React, and TypeScript, the system enforces role-based access and centralized scoring logic. It is optimized for time-critical event usage, allowing administrators to generate and print final rankings immediately after judging concludes.
